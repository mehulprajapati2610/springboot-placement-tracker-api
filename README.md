# Spring Boot Placement Tracker API

A backend project built to strengthen and reinforce core Spring Boot concepts by implementing a real-world Placement Tracker application.

## Tech Stack

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA (Hibernate)
- MySQL
- Maven
- Lombok

## Features

- User Registration & Login
- JWT Authentication & Authorization
- Role-Based Access Control (USER / ADMIN)
- Company CRUD APIs
- Job Application Management
- Global Exception Handling
- RESTful API Design

## Learning Focus

This project was built to strengthen backend fundamentals through hands-on implementation of:

- Spring Security
- JWT Authentication
- JPA & Hibernate
- REST APIs
- Layered Architecture
- DTO Pattern
- Exception Handling
- Role-Based Authorization
